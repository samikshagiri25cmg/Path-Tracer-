# Path-Tracer-

## Description 

This project demonstrates physically-based rendering techniques including multiple light bounces, material scattering, reflections, refraction, and progressive frame accumulation.

### Features

- Real-time path tracing in WGSL
- Multiple ray bounces 
- Lambertian materials
- Metallic reflection
- Glass / dielectric refraction
- Random sampling per pixel
